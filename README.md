# 4919
4919(しょくいく).jp project

## 開発環境
- Vue@2
- gulp (yarn)

## インストール

< node.js を brew で入れている場合 >
```
$ brew install yarn
$ yarn install
```

< 開発時 >
```
$ yarn gulp
```

## ローカル環境上で動作確認をする場合
以下のコマンドを入力する事で、"http://localost:8080/"でブラウザからアクセス可能
```
$ npm install -g live-server
$ cd <ドキュメントルートとするディレクトリ>
$ live-server .
```
